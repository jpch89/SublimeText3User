# SublimeText3User
本仓库存放自用的 Sublime Text 3 的 User 文件夹，以便在多设备之间同步配置。

## 文件夹路径
- `Windows`：`C:\Users\用户名\AppData\Roaming\Sublime Text 3\Packages\User`

## 快捷键一览
由于习惯了 `Jetbrains` 系的 `IDE`，我把 `Sublime Text 3` 的很多快捷键做了改动。
- `Ctrl + d`：复制行
- `Shift + Enter`：在下方新建一行
- `Ctrl + Alt + l`：自动 `PEP 8` 格式化
- `Shift + f10`：运行 `.py` 文件
